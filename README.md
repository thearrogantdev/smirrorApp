# sMirror App

> Browser-based control app for your [sMirror](https://github.com/thearrogantdev/smirrorBackend).

The app runs directly on the mirror — no install needed. Open a browser, connect, done.

## Features

- **Works anywhere** — runs in any modern browser, on any device
- **Multi-mirror support** — connect to and switch between multiple mirrors from one app
- **Instant load** — optimized for fast startup, even on low-bandwidth local networks
- **Web-first** — no App Store, no APK, no setup

## Getting Started

Point your browser at your mirror's local address:

```
http://<mirror-ip>:9002
```

That's it.
